# Finance Tracker

Простое приложение для учёта доходов и расходов на React.

## Функционал:
- ✅ Учёт доходов и расходов
- ✅ Баланс в реальном времени
- ✅ История операций
- ✅ LocalStorage (сохранение данных)
- ✅ Валидация ввода

## Технологии:
- React
- Hooks (useState, useEffect)
- LocalStorage
- CSS-in-JS

## Как запустить:
npm install
npm start
